# Finetune Builder Tooling

This folder contains dataset-construction scripts for canonical SFT corpora and orchestrator seeds.

- `build_verified_sft_dataset.py` builds the verified math+logic corpus.
- `build_training_dataset_0.py` builds the identity/style dataset.
- `build_chunked_sft_dataset.py` creates bounded-context chunked variants.
- `build_orchestrator_dataset.py` builds the orchestrator V1 seed set.
