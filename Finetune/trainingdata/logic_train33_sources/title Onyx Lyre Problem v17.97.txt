title: "Onyx Lyre Problem v17.97"

  description: "At 20:25, the Onyx Lyre vanished from a sealed display in the northern wing. Four individuals (1, 2, 3, 4) had verified badge activity at that time. Exactly one suspect lies in all their statements; the remaining three are fully truthful. Only one is actually guilty. Each suspect gives three statements:

    Suspect 1:
      (1) 'I left the northern wing at 20:15, so I am not involved.'
      (2) 'If 2 is lying, then 4 must have taken the Lyre.'
      (3) '3 was not near the display after 20:20.'

    Suspect 2:
      (1) '1 actually remained until 20:25, contradicting 1’s 20:15.'
      (2) 'I was never close to the Lyre after 20:05.'
      (3) 'If 3 is telling the truth, then 4 is lying entirely.'

    Suspect 3:
      (1) 'I never approached that display after 20:10.'
      (2) 'Either 2 is guilty, or 1 is lying about leaving early—but not both.'
      (3) '4 is surely not the culprit; I saw them leaving at 20:20.'

    Suspect 4:
      (1) '3 definitely came near the display at 20:20, contradicting 3’s denial.'
      (2) 'I am innocent, with logs proving I was in a separate corridor.'
      (3) 'If 2 is lying, then 1 must be correct about 20:15 departure.'

  question: "Which suspect (1, 2, 3, or 4) stole the Onyx Lyre? Provide only a single digit."

answer: >
  explanation: "Formal Deduction with Predicate Logic:

  Let L(x) = 'x lies in all statements,' T(x) = 'x is truthful in all statements,' G(x) = 'x is guilty.'
  Exactly one suspect satisfies L(x)=true, exactly one satisfies G(x)=true, and the rest T(x).

  Case L(1):
    Then 1(1),(2),(3) are false => 'Left 20:15 => not involved' is false => 1 might be present later or guilty; 'If 2 lying => 4 took Lyre' is false => no forced scenario; '3 not near after 20:20' is false => 3 was near. Meanwhile 2,3,4 truthful => 2(1) '1 remained 20:25' must be true => conflict with 3(3) or 4(1) about who was near. Contradiction arises.

  Case L(2):
    Then 2(1),(2),(3) are false =>
    - (1) false => 1 did not remain until 20:25 => 1’s 20:15 claim might be correct
    - (2) false => 2 was near Lyre after 20:05 => suspicious
    - (3) false => if 3 is truthful => 4 is lying entirely is not correct => 4 can be truthful
    Meanwhile 1,3,4 are truthful => 3(2) 'Either 2 is guilty or 1 is lying time not both' => if 1 not lying => 2 is guilty => no contradiction. G(2)=true scenario works.

  Case L(3):
    Then 3(1),(2),(3) false => 3 approached after 20:10, 'Either 2 guilty or 1 lying time not both' fails => might be both or neither, '4 not culprit' is false => 4 can be culprit. Then 1,2,4 are truthful => 1(1) 'Left 20:15 => not involved' => G(1)=false, 2(2) 'Never close after 20:05 => not culprit' => G(2)=false => 4(2) 'I am innocent => G(4)=false'? Contradiction, because we’d have no culprit left. Fails.

  Case L(4):
    Then 4(1),(2),(3) are false => '3 near display 20:20' false => 3 might not be near, 'I am innocent' false => 4 could be culprit, 'If 2 lying => 1 must be correct' also false => confusion. Meanwhile 1,2,3 are truthful => 1(1) 'Left 20:15 => not thief' => G(1)=false, 2(2) 'Never close after 20:05 => G(2)=false, 3(3) '4 is not culprit' is true => conflict since 4 might be culprit. Contradiction.

  Only L(2) scenario is consistent => suspect 2 is lying in all statements => G(2)=true. They stole the Onyx Lyre."

  final: "2"
