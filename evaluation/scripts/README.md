# Evaluation Scripts

This folder contains the concrete PowerShell launchers for the local math-loop evaluation surface.

- `run_math_loop.ps1` solves a single problem through the two-body loop.
- `evaluate_math_loop.ps1` runs manifest-based evaluation.
- `run_math_loop_observer.ps1` launches the Tk observer GUI.

Root scripts remain as thin forwarders for stable CLI entrypoints.
